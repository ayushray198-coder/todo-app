# MERN Todo App

## About Project

This is a simple To-Do List app where users can manage their daily tasks.
Users can add, delete, update, and mark tasks as complete.

---

## Features

* Add new todo
* Delete todo
* Update todo
* Mark as complete or incomplete

---

## Tech Used

* React (Frontend)
* Node.js + Express (Backend)
* MongoDB (Database)

---

## How to Run

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev

## Problems Faced

* Git setup issues
* Deployment errors on Netlify and Render
* File name case sensitivity issue
* MongoDB connection error (wrong connection string)
* Database not connecting due to network access settings
* Express routes not working properly (wrong path or method)
* CORS error while connecting frontend and backend

---

## Solutions

* Fixed Git setup step by step
* Corrected build settings and redeployed
* Fixed file naming (capital/small letters)
* Checked and corrected MongoDB connection string
* Allowed IP access in MongoDB Atlas
* Fixed API routes and HTTP methods
* Enabled CORS in backend server
